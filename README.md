FitZone Landing Page

This project is developed as part of my **Web Development Internship at SkillCraft Technology**.

## 🚀 Task Objective

Create an **interactive navigation menu** that:

* Remains fixed at the top of the page
* Changes style when scrolling
* Includes hover effects for better user interaction

## 💡 Project Overview

I extended the basic requirement by building a **modern gym/fitness landing page (FitZone)** with a premium UI design inspired by real-world websites.

## ✨ Features

* 🔝 Fixed navigation bar
* 🎨 Navbar style change on scroll
* 🖱️ Smooth hover effects
* 💎 Modern split hero section
* 🧊 Glassmorphism UI cards
* 📱 Fully responsive design
* 🎬 Scroll reveal animations
* ⚡ CTA section & professional footer

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
